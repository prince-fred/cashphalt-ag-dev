# Cashphalt Brand Guide: Modern Industrial

> "Functional, High-Contrast, Swiss-Style Efficiency."

This document serves as the single source of truth for the visual Identity of Cashphalt. The design prioritizes outdoor legibility, speed of interaction, and a professional, authoritative aesthetic.

## 1. Core Identity
*   **Keywords:** Precision, Clarity, Infrastructure, Modern, Speed.
*   **Vibe:** A digital extension of modern city infrastructure. Not "under construction," but "expertly paved."
*   **Tone of Voice:** Direct, concise, no-nonsense. "Session Started" instead of "You have successfully started your parking session."

---

## 2. Color Palette
High contrast is non-negotiable for outdoor visibility (glare, night).

### Primary Colors
*   **Signal Yellow**
    *   **Hex:** `#FFD700`
    *   **Usage:** Primary Actions (Pay, Start), Highlights, Active States.
    *   **Text on Yellow:** ALWAYS Black (`#121212`).
*   **Matte Black**
    *   **Hex:** `#121212` (Not pure #000000)
    *   **Usage:** Primary Text, Backgrounds (Inverted sections), Secondary Buttons.
    *   **Text on Black:** White (`#FFFFFF`) or Signal Yellow (`#FFD700`).
*   **White**
    *   **Hex:** `#FFFFFF`
    *   **Usage:** Page Backgrounds, Card Backgrounds (Clean).

### Secondary & Semantic
*   **Concrete Grey**
    *   **Hex:** `#F4F4F5`
    *   **Usage:** Section Backgrounds, Input fields (inactive).
*   **Slate Outline**
    *   **Hex:** `#E4E4E7` (Zinc-200)
    *   **Usage:** Borders, Dividers.
*   **Error Red**
    *   **Hex:** `#DC2626`
    *   **Usage:** Error states, "Time Expired".
*   **Success Green**
    *   **Hex:** `#16A34A`
    *   **Usage:** "Payment Successful", "Session Active".

---

## 3. Typography
**Font Family:** `Inter` (Google Fonts) or system sans-serif.
**Style:** Geometric, clean, high legibility.

### Type Scale
*   **H1 (Page Titles):** 32px / Bold (700) / Tracking: -0.02em / Leading: 1.2
*   **H2 (Section Headers):** 24px / Semi-Bold (600) / Tracking: -0.01em
*   **H3 (Card Headers):** 18px / Medium (500)
*   **Body (Primary):** 16px / Regular (400) / Leading: 1.5
*   **Body (Secondary/Label):** 14px / Medium (500) / Color: `#71717A`
*   **Data/Timer:** 48px+ / Bold / Monospaced (optional for numbers)

---

## 4. UI Components

### Buttons
**Primary Button**
*   **Background:** Signal Yellow (`#FFD700`)
*   **Text:** Matte Black (`#121212`)
*   **Shape:** Rounded Rectangle (8px radius) â€” *Not pill-shaped, not sharp.*
*   **Height:** 56px (Large tap target).
*   **Style:** Flat, no gradient. Subtle active state (95% opacity).

**Secondary Button**
*   **Background:** Transparent
*   **Border:** 2px Solid Matte Black (`#121212`)
*   **Text:** Matte Black (`#121212`)
*   **Shape:** Rounded Rectangle (8px radius).

### Information Cards
*   **Background:** White
*   **Border:** 1px Solid Slate Outline (`#E4E4E7`)
*   **Shadow:** `0px 2px 4px rgba(0,0,0,0.05)` (Very subtle).
*   **Radius:** 12px.
*   **Padding:** 16px or 20px.

### Input Fields
*   **Background:** Concrete Grey (`#F4F4F5`)
*   **Border:** None (Default) / 2px Solid Black (Focus).
*   **Text:** 18px (Large to prevent zooming on iOS).
*   **Radius:** 8px.

### Navigation / Header
*   **Background:** Matte Black (`#121212`)
*   **Text/Icons:** White (`#FFFFFF`)
*   **Logo:** Simple, bold wordmark.
*   **Height:** 64px.

---

## 5. Spacing & Layout
*   **Grid System:** 4-column (Mobile).
*   **Base Unit:** 4px.
*   **Margins:** 20px (standard outer margin).
*   **Gaps:** 16px (between elements).

**Golden Rule:** If in doubt, add more whitespace. The interface must breathe to be readable at a glance.
